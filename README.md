# Cryptocurrency Price Tracker

This application is a cryptocurrency price tracker built using React and Tailwind. It allows users to monitor real-time prices and trends of various cryptocurrencies. With an intuitive user interface, users can easily search for specific cryptocurrencies, view detailed information, and track their favorite coins.

## Key Features

- Real-time price updates for popular cryptocurrencies
- Detailed information and statistics for each cryptocurrency
- Search functionality to find specific coins
- Favorite coin tracking for personalized monitoring
- Responsive design for seamless usage across devices

## Technologies Used

- React v18
- React Router v6
- TypeScript
- WebSockets
- Headless UI
- Tailwind CSS v3
- SCSS for styling

## Description

This project serves as a demonstration of my skills as a frontend developer, showcasing my ability to create a loosely coupled architecture with modern web technologies. By adopting a loosely coupled approach, the application's components are decoupled and can be developed, tested, and maintained independently, resulting in better scalability and flexibility.

## Getting Started

To get started with the Cryptocurrency Price Tracker project, follow these steps:

1. Clone the repository: `git clone https://github.com/amirzarafshani/crypto-price-tracker.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open the application in your browser at `http://localhost:3000`

Feel free to explore the code and provide feedback! Stay up-to-date with the latest cryptocurrency prices and trends with this powerful price tracker.
